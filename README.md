# 🚀 Meus Estudos - 2º Ano do Ensino Técnico

Seja bem-vindo(a) ao meu repositório de estudos! Aqui eu centralizo e organizo todos os códigos, projetos, exercícios práticos e simulações desenvolvidos durante o meu segundo ano, com foco em desenvolvimento de software, banco de dados e eletrônica.

---

## 🛠️ Tecnologias e Linguagens Utilizadas

Durante este ano, o foco está sendo o aprendizado e aprofundamento nas seguintes tecnologias:

*   **PHP:** Desenvolvimento backend, manipulação de dados, processamento de formulários e lógica de servidores (usando ambientes como XAMPP).
*   **C++:** Estrutura de dados, lógica de programação estruturada e conceitos fundamentais de computação.
*   **Java (Swing):** Criação de interfaces gráficas (GUI) e conceitos de Orientação a Objetos.
*   **Kotlin:** Introdução ao desenvolvimento moderno, focado em aplicativos ou lógica orientada a objetos.
*   **SQL:** Criação de bancos de dados, modelagem, relacionamentos e consultas estruturadas (Queries).
*   **HTML5 & CSS3:** Estruturação, estilização de layouts responsivos e design para a web.
*   **Arduino:** Programação de microcontroladores, lógica de automação e integração com componentes físicos (sensores/LEDs).
